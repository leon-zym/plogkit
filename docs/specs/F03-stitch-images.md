# F03 多图拼接

- 状态：已实现
- 关联：[ADR 0003](../adr/0003-document-driven-architecture.md)、[ADR 0007](../adr/0007-export-pipeline.md)

## 概述

将多张照片拼接为竖向或网格构图，自动布局，无需手动摆放。拼接结果仍是统一文档，可继续叠加文字与背景处理。

## 范围

- 竖向拼接与网格拼接两种布局。
- 可调图片间距。
- 背景色。
- 图片顺序调整。

## 非目标

- 自由画布式手动摆位、不规则模板、图片裁剪。

## 需求与场景

### 需求 1：布局

#### Scenario: 竖向拼接

- GIVEN 用户已导入 3 张图片
- WHEN 用户选择竖向拼接
- THEN 预览显示 3 张图片按导入顺序纵向排列，宽度对齐

#### Scenario: 网格拼接

- GIVEN 用户已导入 4 张图片
- WHEN 用户选择网格拼接
- THEN 预览显示 2×2 网格布局，单元格尺寸一致

#### Scenario: 调整图片间距

- GIVEN 处于竖向拼接布局
- WHEN 用户增大图片间距
- THEN 预览即时反映新间距，背景色填充间隙
- AND 滑杆释放时记录一次提交

### 需求 2：顺序调整

#### Scenario: 调整图片顺序

- GIVEN 竖向拼接中有 3 张图片
- WHEN 用户将第 3 张图片移动到第 1 位
- THEN 预览按新顺序重新布局

### 需求 3：统一文档能力叠加

#### Scenario: 拼接结果上加字

- GIVEN 一个竖向拼接构图
- WHEN 用户执行 F01 的加字流程
- THEN 文本块渲染在拼接构图之上
- AND 后续调整拼接间距时文本块位置保持画布坐标不变

## 已解决问题

- 网格固定 2 列，行数随图片数量自适应；奇数张时最后一张占一个等宽单元格。
- MVP 导入与拼接统一最多 9 张，并与导出尺寸上限联动。
