# F02 背景

- 状态：已确认
- 关联：[ADR 0003](../adr/0003-document-driven-architecture.md)

## 概述

为照片设置背景色与输出比例。这是一个快速任务流，不是图层编辑器。

## 范围

- 背景色设置（纯色）。
- 输出比例：1:1、3:4、4:5、9:16、原始比例。

## 非目标

- 边距/内边距、边框、圆角。
- 渐变/图片背景、阴影。

## 需求与场景

### 需求 1：背景色

#### Scenario: 设置背景色

- GIVEN 一个含单张图片的编辑会话
- WHEN 用户选择白色背景
- THEN 预览显示图片周围由白色背景填充（按当前输出比例）
- AND 操作完成时文档模型记录一次提交

### 需求 2：输出比例

#### Scenario: 切换输出比例

- GIVEN 一张 4:3 的横图
- WHEN 用户选择 3:4 输出比例
- THEN 画布变为 3:4，图片按适配规则居中呈现，多余区域由背景色填充

#### Scenario: 恢复原始比例

- GIVEN 画布处于 3:4 比例
- WHEN 用户选择"原始比例"
- THEN 画布恢复为图片原始宽高比

## 开放问题

- 比例切换时图片默认适配策略（contain 或 cover）的最终选择，待交互稿确认。
