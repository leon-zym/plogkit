# F02 边框与背景

- 状态：已确认
- 关联：[ADR 0003](../adr/0003-document-driven-architecture.md)

## 概述

为照片添加边距、边框、圆角与背景色，并设定输出比例。这是一个快速任务流，不是图层编辑器。

## 范围

- 边距/内边距调整。
- 背景色设置（纯色）。
- 简单边框处理。
- 圆角。
- 输出比例：1:1、3:4、4:5、9:16、原始比例。

## 非目标

- 渐变/图片背景、花式边框素材、逐边独立边距、阴影。

## 需求与场景

### 需求 1：边距与背景

#### Scenario: 添加边距与背景色

- GIVEN 一个含单张图片的编辑会话
- WHEN 用户增大边距并选择白色背景
- THEN 预览显示图片四周出现白色边距
- AND 滑杆释放时文档模型记录一次提交

### 需求 2：圆角

#### Scenario: 应用圆角

- GIVEN 图片已有边距与背景
- WHEN 用户调整圆角半径
- THEN 预览中图片四角按半径圆滑，背景不受影响

### 需求 3：输出比例

#### Scenario: 切换输出比例

- GIVEN 一张 4:3 的横图
- WHEN 用户选择 3:4 输出比例
- THEN 画布变为 3:4，图片按适配规则居中呈现，多余区域由背景色填充

#### Scenario: 恢复原始比例

- GIVEN 画布处于 3:4 比例
- WHEN 用户选择"原始比例"
- THEN 画布恢复为图片原始宽高比

## 开放问题

- 比例切换时图片默认适配策略（contain 或 cover）的最终选择，待交互稿确认。
