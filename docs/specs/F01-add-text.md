# F01 为图片加字

- 状态：已确认
- 关联：[ADR 0003](../adr/0003-document-driven-architecture.md)、[ADR 0005](../adr/0005-text-editing-model.md)

## 概述

在文档画布上添加、编辑与摆放文本块。文本用于 plog 叙述，中文长段排版是一等公民。

## 范围

- 添加/编辑/删除文本块；一个文档支持多个文本块。
- 字号、颜色、对齐（左/中/右）、行高；可选文本背景块。
- 一小组预置文本样式（干净、克制）。
- 拖动摆放（画布绝对坐标，可选锚定图块）。

## 非目标

- 装饰性字效（描边、渐变、阴影组合等）、贴纸文本、竖排、富文本混排。

## 需求与场景

### 需求 1：添加与编辑文本

#### Scenario: 添加文本块

- GIVEN 一个含至少一张图片的编辑会话
- WHEN 用户点击"加字"并在输入面板输入「周末的海边日记」后确认
- THEN 画布上出现渲染后的文本块，内容为「周末的海边日记」
- AND 文档模型新增一个文本元素

#### Scenario: 编辑已有文本

- GIVEN 画布上已有一个文本块
- WHEN 用户点击该文本块并在输入面板修改内容后确认
- THEN 画布上该文本块内容更新为修改后的文本

#### Scenario: 提交空文本等效删除

- GIVEN 画布上已有一个文本块
- WHEN 用户进入编辑并清空内容后确认
- THEN 该文本块从画布与文档模型中移除

### 需求 2：中文长段排版

#### Scenario: 长段中文自动换行

- GIVEN 用户在输入面板输入超过画布宽度的连续中文长段（≥100 字）
- WHEN 用户确认提交
- THEN 文本在文本块宽度内自动换行渲染，无字符截断或溢出画布

### 需求 3：样式调整

#### Scenario: 修改字号与颜色

- GIVEN 画布上已有选中的文本块
- WHEN 用户将字号调大并选择新颜色
- THEN 画布渲染即时反映新字号与颜色
- AND 释放控件后文档模型记录一次提交（可撤销）

#### Scenario: 文本背景块

- GIVEN 画布上已有选中的文本块
- WHEN 用户开启文本背景块并选择背景色
- THEN 文本后方渲染出对应颜色的背景块

### 需求 4：摆放

#### Scenario: 拖动文本块

- GIVEN 画布上已有一个文本块
- WHEN 用户拖动该文本块到新位置并松手
- THEN 文本块停留在新位置
- AND 松手时文档模型记录一次位置提交（拖动过程不逐帧入档）
